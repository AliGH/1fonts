1Fonts - Persian web fonts gallery.

1Fonts is a persian fonts gallery for web developers. All fonts optimized and fixed issues for using in web.

Features

  • Easy to use
  • IE6+ support
  • Inclute 3 font types (eot, ttf, woff)

Demo

alimd.github.io/1fonts

How to use

Copy the code as the first element in the <head> of your HTML document.

<link href='http://alimd.github.io/libs/1fonts/1/1fonts.css' rel='stylesheet' type='text/css' />
